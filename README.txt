Рассылка писем
Описание:

Функция send_email позволяет отправлять электронные письма. Она принимает 2 обычных аргумента: сообщение и получатель, а также один обязательно именованный аргумент со значением по умолчанию — отправитель.

Как использовать:

Создайте файл с функцией send_email.
Импортируйте необходимые библиотеки.
Реализуйте логику функции в соответствии с описанием.
Протестируйте функцию.
Отправьте проект на GitHub.
Поделитесь ссылкой на проект с другими разработчиками.
При необходимости предоставьте дополнительную документацию или примеры использования.

Обязательные аргументы:

message — сообщение, которое будет отправлено.
recipient — адрес получателя.
Именованный аргумент:

sender — адрес отправителя (по умолчанию «default@example.com»).
Логика функции:

Проверка на корректность e-mail отправителя и получателя.
